# CodeAlpha Task 2 - Unemployment Analysis with Python

## 📌 Project Overview
This project is completed as part of the **CodeAlpha Data Science Internship**. The objective of this project is to analyze unemployment data using Python by performing data cleaning, exploratory data analysis (EDA), and data visualization to identify trends and the impact of COVID-19 on unemployment rates.

---

## 📊 Project Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Missing Value Handling
- Duplicate Data Removal
- Unemployment Trend Analysis
- State-wise Average Unemployment Analysis
- COVID-19 Impact Analysis
- Monthly Unemployment Trend Analysis
- Data Visualization

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

---

## 📂 Dataset
**Dataset:** `Unemployment in india.csv`

---

## 📈 Visualizations
- Unemployment Rate Over Time
- State-wise Average Unemployment
- COVID-19 Impact on Unemployment
- Distribution of Unemployment Rate
- Unemployment Heatmap
- Monthly Average Unemployment Rate

---

## 📋 Project Workflow
1. Import required libraries
2. Load the dataset
3. Explore the dataset
4. Clean the data
5. Analyze unemployment trends
6. Visualize the data
7. Generate insights and conclusion

---

## 📌 Conclusion
The unemployment dataset was successfully analyzed using Python. The analysis showed the impact of COVID-19 on unemployment rates, highlighted regional differences, and identified monthly trends through data visualization. These insights can help in understanding employment patterns and support better decision-making.
